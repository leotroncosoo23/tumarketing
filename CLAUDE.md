# Contexto del Proyecto
Plataforma Full Stack orientada a marketing digital, venta de cursos, servicios, recursos y blogs. Incluye un panel de administración y un portal de alumnos.

# Stack Tecnológico Principal
- Frontend: React, Next.js
- Backend: Node.js
- Base de Datos: SQL

# Reglas Generales de Desarrollo
- Mantener un código limpio, modular y escalable.
- Escribir código en inglés (variables, funciones) pero comentarios y textos de UI en español.
- Antes de escribir o refactorizar código complejo, planificar la solución paso a paso.
- No asumir lógica de negocio; si hay ambigüedad en los modelos de datos o el flujo de pago, preguntar primero.
